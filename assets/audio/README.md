# 🎵 Assets de Audio - Expediente Kōrin

## 📁 Estructura de Carpetas

```
assets/audio/
├── music/          # Música de fondo (loops)
└── sfx/            # Efectos de sonido (cortos)
```

## 🎼 Archivos Necesarios

### **music/** - Música de Fondo
Coloca aquí los archivos de música ambiente:

- `login_ambient.mp3` - Atmósfera de fondo para pantalla de login
  - **Buscar**: "dark ambient", "horror atmosphere", "japanese horror music"
  - **Duración**: 1-3 minutos (loop)
  - **Volumen recomendado**: 15-25%
  - **Formato**: MP3 o OGG

### **sfx/** - Efectos de Sonido
Coloca aquí los efectos cortos:

#### Efectos del Título:
- `glitch_01.mp3` - Efecto glitch VHS/TV
  - **Buscar**: "glitch sound", "vhs static", "tv interference"
  - **Duración**: 0.1-0.3 segundos
  - **Volumen recomendado**: 30-40%

#### Efectos de Botones:
- `button_hover.mp3` - Sonido al pasar el mouse sobre botón
  - **Buscar**: "ui hover", "button hover dark", "subtle whoosh"
  - **Duración**: 0.05-0.1 segundos
  - **Volumen recomendado**: 20-30%

- `button_click.mp3` - Sonido al hacer click en botón
  - **Buscar**: "button click", "typewriter key", "ui click dark"
  - **Duración**: 0.1-0.2 segundos
  - **Volumen recomendado**: 40-50%

## 📝 Notas

- Todos los archivos deben ser **libres de derechos** o con licencia Creative Commons
- Formatos recomendados: **MP3** (mejor compatibilidad) u **OGG** (mejor calidad/tamaño)
- Mantén los archivos pequeños (< 1MB para SFX, < 5MB para música)
- Nombra los archivos en **snake_case** (minúsculas con guiones bajos)
